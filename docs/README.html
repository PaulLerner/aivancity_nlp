aivancity NLP#

Main repository for the 2024-2026 Natural Language Processing class at aivancity by Paul Lerner (both during semester 1 and 2)

Classes (2025-2026)#

  • Introduction to NLP and Text Classification

  • Distributional Semantics

  • N-gram and Neural Language Models: from Shannon to ChatGPT

Classes (2024-2025)#

  • LLM Architectures: Attention Mechanism and Transformers

  • Benchmarking / Ethical, social, and environmental issues
